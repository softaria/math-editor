withLimits

  • withLimits(): this
    • Defined in standard-edition/renderer/katex/tree/op.ts:112
    description

    Analogue of latex's '\limits' in Katex; we want limit to be in display mode in all places including fraction, but in plain latex it is in inline mode in fraction; normal way to achieve that is '\limits', but it is not presented in Katex tree and parsed explicitly by Katex, so we have to use details of Katex's implementation.

    Returns this
