Class NonTerminalInsertVisitor

Этот класс по литералу и isLeft (по сути - по NonTerminalPointer) умеет находить позиции ввода внутри каждого узла Позиции бывают - слева от узла, справа от узла и где-то внутри узла. Если позиция где-то внутри, то он находит child-а, с ней связанного
